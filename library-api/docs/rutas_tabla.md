# Tabla de Endpoints

| Método | Ruta                               | Descripción                         |
|--------|-------------------------------------|-------------------------------------|
| GET    | /users                              | Obtener usuarios                    |
| POST   | /users                              | Crear usuario                       |
| PUT    | /users/:id                          | Actualizar usuario                  |
| DELETE | /users/:id                          | Eliminar usuario                    |
| GET    | /users/leaderboard                  | Top 10 puntajes                     |
| GET    | /levels                             | Obtener niveles                     |
| POST   | /levels                             | Crear nivel                         |
| PUT    | /levels/:id                         | Actualizar nivel                    |
| DELETE | /levels/:id                         | Desactivar nivel                    |
| GET    | /sessions                           | Listar sesiones                     |
| GET    | /sessions/:id                       | Sesión por ID                       |
| POST   | /sessions                           | Crear sesión                        |
| PUT    | /sessions/:id                       | Actualizar sesión                   |
| GET    | /sessions/highscore/:levelId        | Puntaje más alto de un nivel        |
| POST   | /login                              | Login de usuario                    |
